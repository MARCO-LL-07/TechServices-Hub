# TechServices Hub

Sistema de gestión de alquiler de vehículos desarrollado con **Node.js, Express y Sequelize**, conectado a una base de datos PostgreSQL.

---

## Descripción del proyecto

TechServices Hub permite gestionar usuarios, carros, categorías, alquileres y pagos.  
Incluye funcionalidades avanzadas como:

-  Gestión de usuarios 
- CRUD de carros
- Categorías de vehículos 
- Alquiler de vehículos
- Registro de pagos
- Soft Delete 
- Relaciones entre tablas

## Gestión de Conflictos

### Conflictos
- El repositorio remoto tenía una estructura diferente a la del proyecto local.
- Git detectó que ambos repositorios tenían historiales independientes.
### Soluciones
- Se revisaron manualmente los archivos y se conservaron las versiones del proyecto local, eliminando duplicados innecesarios.
- Se utilizó el comando `git pull origin main --allow-unrelated-histories` para permitir la fusión de ambos repositorios y se mantuvo la estructura local del backend.
